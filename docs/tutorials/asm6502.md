# 6502 Assembly Tutorial

## Introduction

The MOS 6502 (1975) was one of the most influential microprocessors in history, powering the Apple II, Commodore 64, Atari 2600, and original Nintendo Entertainment System. Learning 6502 assembly teaches the fundamentals of how computers actually work at the hardware level.

**Key characteristics:**
- 8-bit data, 16-bit address space (64KB)
- Three registers: **A** (accumulator), **X** and **Y** (index)
- Stack pointer **SP** (hardwired to page 1: $0100–$01FF)
- 6 status flags: **N**egative, o**V**erflow, **B**reak, **D**ecimal, **I**nterrupt, **Z**ero, **C**arry
- Simple RISC-like instruction set (~56 opcodes)

## Memory Map

```
$0000–$00FF   Zero Page  (fast 1-byte addressing)
$0100–$01FF   Stack      (hardwired, grows downward)
$0200–$FFFF   General RAM and ROM
$FFFC–$FFFD   Reset vector (startup address)
$FFFE–$FFFF   IRQ/BRK vector
```

## Registers

| Register | Size | Purpose |
|----------|------|---------|
| A | 8-bit | Arithmetic and memory data |
| X | 8-bit | Index register, counter |
| Y | 8-bit | Index register, counter |
| SP | 8-bit | Stack pointer (offset into $01xx) |
| PC | 16-bit | Program counter |
| P | 8-bit | Status flags (NVBDIZC) |

## Hello World

```asm
    .org $0600       ; start address

START:
    LDX #0           ; X = 0 (string index)
LOOP:
    LDA MSG,X        ; load byte from MSG+X
    BEQ DONE         ; branch if zero (end of string)
    STA $F001        ; write to output device
    INX              ; X++
    JMP LOOP         ; repeat
DONE:
    BRK              ; stop execution

MSG:
    .byte "Hello, World!", $0A, 0   ; string with newline, null terminator
```

## Load and Store

```asm
    ; Load immediate (literal value)
    LDA #$42         ; A = 0x42 = 66
    LDX #10          ; X = 10
    LDY #$FF         ; Y = 255

    ; Store to memory
    STA $1000        ; memory[$1000] = A
    STX $1001        ; memory[$1001] = X
    STY $1002        ; memory[$1002] = Y

    ; Load from memory
    LDA $1000        ; A = memory[$1000]

    ; Transfer between registers
    TAX              ; X = A
    TXA              ; A = X
    TAY              ; Y = A
    TYA              ; A = Y
```

## Arithmetic

```asm
    ; Add with carry — always CLC before an add
    LDA #5
    CLC
    ADC #3           ; A = 5 + 3 = 8

    ; Subtract with borrow — always SEC before a subtract
    LDA #10
    SEC
    SBC #3           ; A = 10 - 3 = 7

    ; Increment / decrement
    INX              ; X++
    DEX              ; X--
    INY              ; Y++
    DEY              ; Y--
    INC $10          ; memory[$10]++
    DEC $10          ; memory[$10]--
```

## Comparisons and Branching

```asm
    ; Compare sets flags without changing A
    LDA #5
    CMP #10          ; sets N, Z, C flags

    ; Branch instructions (signed relative offset ±127)
    BEQ EQUAL        ; branch if Z=1 (equal)
    BNE NOTEQUAL     ; branch if Z=0 (not equal)
    BCC BELOW        ; branch if C=0 (unsigned less than)
    BCS ABOVEOREQ    ; branch if C=1 (unsigned >=)
    BMI NEGATIVE     ; branch if N=1 (negative)
    BPL POSITIVE     ; branch if N=0 (positive/zero)

EQUAL:
    LDA #1
    JMP DONE
NOTEQUAL:
    LDA #0
DONE:
    BRK
```

## Loops

```asm
    ; Count from 1 to 10 using X
    LDX #1
LOOP:
    ; ... do something with X ...
    INX
    CPX #11          ; compare X to 11
    BNE LOOP         ; loop if X != 11

    ; Count down from 5 to 0 using DEX/BNE
    LDX #5
COUNTDOWN:
    ; ... body ...
    DEX
    BNE COUNTDOWN    ; loop while X != 0
```

## Subroutines

```asm
START:
    JSR PRINT_HI     ; call subroutine (pushes return addr to stack)
    JSR PRINT_HI
    BRK

PRINT_HI:
    LDA #'H'
    STA $F001
    LDA #'i'
    STA $F001
    LDA #$0A         ; newline
    STA $F001
    RTS              ; return (pops return addr from stack)
```

## Addressing Modes

| Mode | Example | Meaning |
|------|---------|---------|
| Immediate | `LDA #$42` | Literal value |
| Zero page | `LDA $10` | Memory at $0010 |
| Zero page,X | `LDA $10,X` | Memory at $0010+X |
| Absolute | `LDA $1234` | Memory at $1234 |
| Absolute,X | `LDA $1000,X` | Memory at $1000+X |
| Absolute,Y | `LDA $1000,Y` | Memory at $1000+Y |
| Indirect | `JMP ($FFFC)` | Jump via pointer |
| (Indirect,X) | `LDA ($10,X)` | Indexed indirect |
| (Indirect),Y | `LDA ($10),Y` | Indirect indexed |

## Complete Example: Fibonacci

```asm
    .org $0600

    ; Compute first 8 Fibonacci numbers: 1, 1, 2, 3, 5, 8, 13, 21
    LDA #1
    STA $00          ; F(0) = 1
    STA $01          ; F(1) = 1

    LDX #2           ; index = 2
LOOP:
    LDA $00          ; A = F(n-2)
    CLC
    ADC $01          ; A = F(n-2) + F(n-1)
    STA $02          ; F(n) = A

    LDA $01
    STA $00          ; shift: F(n-2) = F(n-1)
    LDA $02
    STA $01          ; shift: F(n-1) = F(n)

    INX
    CPX #8
    BNE LOOP

    BRK              ; F(7) = 21 is now in $01
```

## Quick Reference

| Instruction | Operation |
|-------------|-----------|
| `LDA/LDX/LDY` | Load register |
| `STA/STX/STY` | Store register |
| `TAX/TXA/TAY/TYA` | Transfer registers |
| `ADC` | Add with carry (CLC first) |
| `SBC` | Subtract with borrow (SEC first) |
| `INC/DEC` | Increment/decrement memory |
| `INX/DEX/INY/DEY` | Increment/decrement register |
| `CMP/CPX/CPY` | Compare (sets flags) |
| `BEQ/BNE` | Branch equal/not-equal |
| `BCC/BCS` | Branch carry clear/set |
| `BMI/BPL` | Branch minus/plus |
| `JMP` | Unconditional jump |
| `JSR` | Call subroutine |
| `RTS` | Return from subroutine |
| `PHA/PLA` | Push/pull A to/from stack |
| `NOP` | No operation |
| `BRK` | Software interrupt / stop |
| `CLC/SEC` | Clear/set carry flag |
