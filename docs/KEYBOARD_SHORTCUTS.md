# Keyboard Shortcuts Reference

Complete keyboard shortcut reference for **Time Warp Studio**.
Press **Ctrl+?** inside the IDE to open the searchable shortcuts overlay.

---

## File Operations

| Shortcut | Action |
|----------|--------|
| Ctrl+N | New file |
| Ctrl+O | Open file |
| Ctrl+S | Save file |
| Ctrl+Shift+S | Save as… |
| Ctrl+W | Close tab |
| Ctrl+Z | Undo |
| Ctrl+Shift+Z | Redo |

---

## Execution

| Shortcut | Action |
|----------|--------|
| F5 | Run program |
| Ctrl+F5 | Run with input |
| Shift+F5 | Stop execution |
| F9 | Toggle breakpoint |

---

## Debugging

| Shortcut | Action |
|----------|--------|
| F10 | Step over |
| F11 | Step into |
| Shift+F11 | Step out |

---

## Editor

| Shortcut | Action |
|----------|--------|
| Ctrl+/ | Toggle line comment |
| Ctrl+D | Duplicate line |
| Ctrl+G | Go to line |
| Ctrl+F | Find |
| Ctrl+H | Find & replace |
| Ctrl+L | Select line |
| Tab | Increase indent |
| Shift+Tab | Decrease indent |
| Ctrl+] | Indent selection |
| Ctrl+[ | Outdent selection |
| Ctrl+Space | Trigger autocomplete |

---

## View

| Shortcut | Action |
|----------|--------|
| Ctrl++ | Increase font size |
| Ctrl+- | Decrease font size |
| Ctrl+Shift+P | Command palette |
| Ctrl+? | Keyboard shortcuts overlay |
| Ctrl+Shift+F11 | Focus / distraction-free mode |

---

## Help

| Shortcut | Action |
|----------|--------|
| F1 | Context-sensitive language help |

---

## Canvas (Turtle Graphics)

| Action | Description |
|--------|-------------|
| Scroll wheel | Zoom in / out |
| Click + drag | Pan the canvas |
| 💾 button | Save canvas as PNG |
| ⋯ button | Toggle coordinate grid overlay |

---

## Tips

- Most shortcuts are also available via the menu bar.
- The **Ctrl+?** overlay is searchable — start typing to filter.
- **F1** shows help for the currently active language; if no specific help exists it falls back to the User Manual.
