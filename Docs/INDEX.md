## What is Time Warp IDE?

Time Warp IDE is a unique educational programming environment that brings together six classic programming languages in one unified platform:

- **BASIC** - The classic beginner's language
- **PILOT** - Designed specifically for teaching
- **Logo** - Learn programming through turtle graphics
- **Pascal** - Structured programming fundamentals
- **Prolog** - Logic programming and AI concepts
- **C** - Low-level programming and system concepts

### Why Time Warp?

**Educational Focus**: Every feature is designed with learning in mind. Clear error messages, interactive graphics, and immediate feedback help students understand programming concepts.

**Unified Environment**: Switch between languages seamlessly. Learn different programming paradigms without learning different tools.

**Turtle Graphics**: Built-in graphics make programming visual and fun. See your code come to life with colorful drawings.

**IoT Ready**: Connect to Arduino, Raspberry Pi, and other hardware. Bring your programs into the physical world.

**Cross-Platform**: Runs on Linux, macOS, and Windows. Same experience on every platform.
