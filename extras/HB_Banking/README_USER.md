# HB Banking ERP — User Guide

## What is HB Banking?
A full-featured banking system for managing accounts, transactions, and balances. Use it as a standalone app or launch from Time Warp Studio.

## Getting Started
1. Run `python main.py` to start the ERP suite.
2. Log in with your username and password.
3. Use the menu to:
   - View account balances
   - Deposit, withdraw, or transfer funds
   - Open or close accounts (admin only)
   - View transaction history

## Security
- Each user has a unique login and role (admin, teller, etc.)
- Never share your password

## Maintenance
- Contact your administrator for account or system issues
- Data is backed up regularly by the admin team

## Using with Time Warp Studio
- Ask your admin to enable Studio integration
- Studio can launch and maintain the ERP system

## Support
- For help, contact your system administrator or refer to the admin guide
