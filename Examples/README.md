# Time Warp Studio - Example Programs

## Overview

This directory contains 100+ example programs demonstrating all supported languages in Time Warp Studio. Each example is verified to work with the current implementation and includes comments explaining the code.

**Quick Start:** Press **Ctrl+E** in the IDE or go to **File ‚Üí Examples** to browse and load examples interactively.

---

## üöÄ How to Use Examples

### In Time Warp Studio (Easiest)
1. Launch the IDE: `python Platforms/Python/time_warp_ide.py`
2. Press **Ctrl+E** or go to **File ‚Üí Examples**
3. Browse examples by language and difficulty
4. Click **Load** to open in editor
5. Press **Ctrl+R** or click **Run**

### Manual Method
1. Open **File ‚Üí Open** (Ctrl+O)
2. Navigate to `Examples/[language]/`
3. Select `.bas`, `.logo`, `.pilot`, `.c`, `.pas`, `.pl`, or `.f` file
4. Press **Ctrl+R** to run

## üìñ Command Reference

### Logo Graphics Commands

**Movement:**
```logo
FORWARD n / FD n        ; Move forward n pixels
BACK n / BK n           ; Move backward n pixels
LEFT n / LT n           ; Turn left n degrees
RIGHT n / RT n          ; Turn right n degrees
HOME                    ; Return to center
SETXY x y               ; Move to position (x, y)
```

**Pen Control:**
```logo
PENUP / PU              ; Stop drawing (lift pen)
PENDOWN / PD            ; Start drawing (lower pen)
SETPENCOLOR name/RGB    ; Set color (RED, #FF0000, 255 0 0)
SETPENWIDTH n           ; Set pen thickness in pixels
SETBGCOLOR color        ; Set background color
```

**Control Flow:**
```logo
REPEAT n [commands]     ; Repeat commands n times
TO name [params] ... END ; Define procedure
HIDETURTLE / HT         ; Hide turtle cursor
SHOWTURTLE / ST         ; Show turtle cursor
PRINT [text]            ; Output text to console
```

### BASIC Commands

**I/O:**
```basic
PRINT expr              ; Display output
INPUT var               ; Get user input
PRINT "Hello"; expr     ; Multiple outputs
```

**Variables:**
```basic
LET var = expr          ; Assign value
DIM array(size)         ; Declare array
DIM matrix(10,10)       ; Multi-dimensional array
```

**Control:**
```basic
IF condition THEN ...   ; Conditional execution
FOR i = 1 TO 10 ... NEXT ; Loop with counter
WHILE condition ... WEND ; While loop
DO ... LOOP UNTIL cond  ; Do-until loop
SUB name(...) ... END SUB ; Procedure
FUNCTION name(...) ... END ; Function
```

### PILOT Commands

**I/O:**
```pilot
T: text                 ; Output text (Type)
A: answer               ; Accept user input
C: comment text         ; Comment line
```

**Computation:**
```pilot
COMPUTE var expression  ; Calculate values
E: expression           ; Evaluate expression
```

**Control:**
```pilot
*LABEL                  ; Mark jump target
MATCH pattern / WRONG   ; Conditional branching
JUMP *label             ; Unconditional jump
JUMPT *label            ; Jump if true
LOOP [max-count] ...    ; Loop structure
```
