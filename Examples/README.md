# Time Warp Studio - Example Programs

## Overview

This directory contains **86 example programs** across all 7 supported languages in Time Warp Studio. Each example is verified to work with the current implementation and includes comments explaining the code.

**Quick Start:** Press **Ctrl+E** in the IDE or go to **File → Examples** to browse and load examples interactively.

---

## How to Use Examples

### In Time Warp Studio (Recommended)
1. Launch the IDE: `python Platforms/Python/time_warp_ide.py`
2. Press **Ctrl+E** or go to **File → Examples**
3. Browse examples by language and difficulty
4. Click **Load** to open in editor
5. Press **Ctrl+R** or click **Run**

### Manual Method
1. Open **File → Open** (Ctrl+O)
2. Navigate to `Examples/[language]/`
3. Select a file (`.bas`, `.logo`, `.pilot`, `.c`, `.pas`, `.pl`, or `.f`)
4. Press **Ctrl+R** to run
