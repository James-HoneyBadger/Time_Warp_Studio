## Troubleshooting

### Marketplace Issues
- **Plugin not found**: Check marketplace database connection
- **Installation fails**: Verify plugin_path and permissions
- **Version conflict**: Review version compatibility matrix

### Debugger Issues
- **Breakpoint not hit**: Verify execution path reaches breakpoint
- **Watch not updating**: Check variable scope
- **Profiler overhead**: Reduce sample rate if impacting performance

### AI Issues
- **Suggestions not appearing**: Verify engine initialization
- **Low confidence scores**: Review pattern definitions
- **Performance slow**: Check file size and pattern complexity
