# Time Warp IDE for Python

This is the Python version of the Time Warp IDE (v3.0.0), built with Python and tkinter. It provides a standalone graphical application for editing, running, and visualizing Time Warp scripts.

## Features

- Python GUI (tkinter)
- Turtle graphics canvas
- Live console output
- File operations (Open, Save, Run)
- Built-in examples
- Simple and lightweight

## Prerequisites

- Python 3.8+
- tkinter (usually included with Python)

## Installation

```bash
cd Python
pip install -r requirements.txt
```

## Run

```bash
cd Python
make run
```

Or directly:

```bash
python -m time_warp.app
```

## Project Structure

- `time_warp/` — Python package with application source
- `requirements.txt` — Python dependencies
- `requirements-dev.txt` — Development dependencies
- `Makefile` — Build and run commands
